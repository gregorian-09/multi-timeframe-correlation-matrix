# Mermaid Sources

This folder contains source `.mmd` files used to produce SVG architecture/UML diagrams for contributors.

## Files

- `uml_class.mmd`
- `uml_seq_timer.mmd`
- `uml_seq_right_click.mmd`
- `uml_activity_alerts.mmd`
- `structure_dependencies.mmd`
- `structure_runtime_flow.mmd`

## Render to SVG

From repository root:

```bash
./Docs/scripts/render_svgs.sh
```

This writes SVG outputs to `Docs/assets/`.
